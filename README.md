# EATWHAT

**基于 FreeRTOS 的嵌入式多媒体音频系统设计**

**Audio-player based on FM33FK5410 MCU &amp; FreeRTOS**

“复微杯”全国大学生电子设计大赛全国总决赛参赛作品，开发环境为 Windows 11 + Keil 5。仓库包含完整项目工程以及相关材料。

## 项目源码相关目录：
 - *Driver*：包含有项目中所开发/移植的硬件驱动；
 - *Inc*：项目所包含的.h头文件；
 - *Middlewares​*：涉及到中间件的移植和驱动适配；
 - *Src*​：包含所有.c源文件，实现上层业务功能逻辑。

## 总决赛项目答辩相关材料：
 - *软硬件赛道硬件命题-晚饭吃什么-王政.pdf*：总决赛答辩演示文稿
 - *软硬件赛道硬件命题-晚饭吃什么-王政.mp4*：总决赛现场演示视频
