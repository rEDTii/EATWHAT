# EATWHAT

**基于 FreeRTOS 的嵌入式多媒体音频系统设计**

**Audio-player based on FM33FK5410 MCU &amp; FreeRTOS**

“复微杯”全国大学生电子设计大赛全国总决赛参赛作品，开发环境为 Windows 11 + Keil 5。仓库包含完整项目工程以及相关材料。

## 项目介绍

本项目来自“复微杯”2025全国大学生电子设计大赛选题 - 基于FM33XX系列MCU的多媒体音频终端系统，旨在为复旦微电子集团的全新 MCU 型号探索应用场景与实现方案。

本项目是基于 ARM Cortex-M33 架构 FM33FK5410 MCU 独立开发的嵌入式多媒体系统，集高清音频采集/播放、大容量存储管理、图形化交互于一体。采用 FreeRTOS 实时操作系统，构建了涵盖底层核心驱动、OS依赖驱动、中间件（FATFS、µGUI）及应用层的分层及模块化设计的软件架构 ，集成电容触摸 LCD 屏、双声道 I2S 麦克风与功放、SD 卡存储、外部 flash 存储等硬件外设，实现了具备 GUI 交互的双声道 24-bit 采样的自适应采样率（8-32KHz）高保真 WAV 格式音频录制、播放、闹钟、时钟等功能的完整终端。

项目的技术重点在于高性能高鲁棒的音频流处理、低延迟多任务调度与基于新型号 MCU 实现稳定的外设驱动移植、调试与开发。通过深度优化 SPI DMA 与 I2S DMA，构建了双缓冲乒乓操作的高效稳定数据流通路；针对爆破音问题研究并部署了一种变化率约束的自适应限幅音频流滤波，实测可有效抑制突发瞬态噪声；移植并优化了 FATFS 和 μGUI 库，实现高效内部文件管理与资源优化；设计了基于任务通知和信号量的轻量级任务同步机制，确保了多任务实时响应与稳定性。大量实测与答辩现场演示均证明该系统成功在低功耗MCU上兼顾实现了高带宽多媒体流处理与实时任务响应，具备极高的稳定性和实时性，是嵌入式软硬件协同设计、实时系统应用及多媒体信号处理的综合实践。
