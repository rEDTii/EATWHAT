## 项目介绍

项目来自“复微杯”2025全国大学生电子设计大赛选题 - 基于FM33XX系列MCU的多媒体音频终端系统，旨在为复旦微电子集团的全新 MCU 型号探索应用场景与实现方案。

### 核心功能

- **高保真音频处理**：支持 24-bit 采样深度，自适应采样率（8kHz - 32kHz），WAV 格式高保真录制与播放；
- **图形化交互系统**：基于 μGUI 移植优化的交互界面，支持电容触摸屏操作；
- **智能终端应用**：集成实时时钟、多功能闹钟、文件资源管理等终端功能；
- **大容量存储**：支持 SD 卡（FATFS 文件系统）与外部 Flash 存储，实现音频数据的高效持久化。

### 项目主要亮点

### 1. 高带宽音频流乒乓处理机制
SPI DMA + I2S DMA 的双 DMA + 双缓冲乒乓结构协作配合实现音频数据流的高效存取，支撑双声道多采样率 24-bit WAV 高清音频的录制/播放；SPI DMA 基于信号量实现，降低了CPU 占用，提高任务调度的实时性；

### 2. 自适应限幅滤波算法
基于前值保持的变化率约束的动态限幅，有效抑制突发瞬态噪声的同时保证了实时性，对缓解扬声器爆破音问题具有显著效果；

### 3. 任务调度与轻量级同步互斥设计
- 基于任务通知机制和中断实现触摸模块，通过任务通知值奇偶性判决触摸动作，实现高响应、低功耗的中断式触摸驱动；
- 针对录制/播放与突发闹钟之间的硬件资源竞态问题，设计了相应的业务切换逻辑，保证了文件完整性与系统实时稳定运行；

### 4. 文件管理与 GUI 显示优化
- 在 FATFS 的基础上引入动态链表机制，实时监听文件系统事件并同步文件变化，实现文件高效管理；
- 二次开发 μGUI 图形库，包括显示驱动优化、组件功能拓展、字库访问优化等，保证高效渲染与节省资源的同时提升用户友好性。
